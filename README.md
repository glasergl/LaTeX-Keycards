# LaTeX Keycards
This LaTeX template implements the following requirements:
- Create keycards to learn for any topic 
- All keycards in a single PDF file, e.g., to be used for searching with `Ctrl + F`
- Keycards printable to be used to learn without any digital screens

## Usage
1. Download the repository content and follow the guide in `main.tex`.
1. Print the PDF as A4 format without double page printing
1. Cut with scissors along the lines
